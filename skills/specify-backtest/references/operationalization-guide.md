# 操作化指南:把判断词翻译成可计算谓词

## 原则

1. **每条规则要么翻成谓词,要么进日志。** 不允许在 YAML 里留散文。
2. **优先用书中已给的明确参数。** 书说"量能 2 倍"就写 `volume > 2 * avg(volume, 4)`,不要自己换阈值。
3. **判断词要操作化,并记风险。** "RS 改善"-> 需要选一个可计算定义(如 `RS > RS_4wk_ago` 或 `RS_slope > 0`),选择本身是保真度风险,记日志。
4. **不可计算元素排除,不编码。** 头肩底、趋势线手绘、整数心理等,进日志标"排除",可建议代理变量但标「代理」。
5. **框架无关。** 用通用算子(SMA/EMA/slope/rolling_max/rolling_min/ratio/ATR/stddev/cross_above/cross_below),不绑特定框架 API。

## 常见判断词的参考操作化

| 判断词 | 参考谓词 | 备注 |
|---|---|---|
| "均线上行" | `slope(MA, N) > 0` | N 选 4-8,记风险 |
| "均线走平" | `abs(slope(MA, N)) < eps` | eps 需定,记风险 |
| "突破阻力" | `close > rolling_max(high, N)` | 阻力是区间,N 是选择,记风险(高) |
| "回踩缩量" | `volume < 0.5 * avg(volume, N)` | 系数是选择 |
| "RS 改善" | `RS_slope > 0` 或 `RS > RS_{t-N}` | 选哪个是风险 |
| "放量突破" | `volume > 2 * avg(volume, N)` | 若书明确给倍数则用书的 |
| "跌破支撑" | `close < rolling_min(low, N)` | 同阻力 |
| "形态"(头肩底等) | 排除 | 不可直接计算 |

## 保真度风险分级

- **低**:书里给了明确参数/公式,直接用。
- **中**:判断词有合理操作化,但选择有偏差空间(如窗口 N)。
- **高**:操作化可能与原意显著偏离(如"阻力"用滚动最高近似),或元素被排除/代理。

高风险行必须在日志中醒目标注,供用户审阅确认。

## 操作化日志格式

| 书中原话 | 回测谓词 | 保真度风险 | 备注 |
|---|---|---|---|
| "30 周均线(加权)" | `SMA(close,30)` | 中 | 简单 vs 加权,有滞后差 |
| ... | ... | ... | ... |

## 不可计算元素的处理

- 默认:排除 + 日志记"排除"。
- 若给代理变量:标「代理」,记保真度风险为"高",并说明代理与原元素的差异。
- 不要静默丢弃任何书中规则--所有规则要么进 YAML,要么进日志。
